unfill
======

Functions providing the inverse of Emacs' fill-paragraph and fill-region

### Installation

Preferred method: install from [MELPA](http://melpa.org/).

Otherwise:

    (add-to-list 'load-path "/dir/containing/unfill")
    (require 'unfill)

### Usage

    M-x unfill-region
    M-x unfill-paragraph
    M-x unfill-toggle
